# Sistema de Prevención de Inundaciones CDMX

## Descripción
Sistema web para predecir riesgo de inundación basado en datos históricos de probabilidades por alcaldía, lluvia en tiempo real y respuestas del usuario.

- Python
- Flask
- Pandas
- Folium
- OpenWeather API

## Instalación
1. `pip install -r requirements.txt`
2. `python app.py`
3. Visita http://localhost:5000
